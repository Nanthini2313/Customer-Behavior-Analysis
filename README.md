📊 Customer Behavior Dashboard


📌 Overview

The Customer Behavior Dashboard is an interactive data visualization project built using Microsoft Power BI. It analyzes customer purchasing patterns, preferences, and engagement to generate actionable business insights.


🎯 Objectives

Understand customer buying behavior
Identify high-value customers
Analyze sales and revenue trends
Improve customer retention strategies


📁 Dataset

The dataset contains 3,900 customer records with multiple features capturing demographics and purchasing behavior.

🔹 Key Features

Customer ID, Age, Gender

Item Purchased & Category

Purchase Amount (USD)

Review Rating

Subscription Status

Shipping Type

Discount & Promo Code Usage

Previous Purchases

Payment Method

Purchase Frequency


🛠️ Tools & Technologies

1.Microsoft Power BI

2.Power Query (Data Cleaning & Transformation)

3.DAX (Data Analysis Expressions)


💡 Key Insights

Clothing category generates the highest revenue and sales

Majority of customers are non-subscribers

Young adults contribute the most to revenue

Customer engagement varies across demographics and categories


🚀 How to Use

Download the .pbix file from this repository
.Open it using Microsoft Power BI
.Use filters (slicers) to explore different insights
.Interact with visuals to analyze customer behavior


📈 Conclusion

This dashboard provides a clear understanding of customer behavior and supports data-driven decision-making. It helps businesses optimize marketing strategies, enhance customer experience, and increase revenue.
